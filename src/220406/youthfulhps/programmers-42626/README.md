### 문제 풀이 기록

1. 최소힙을 사용한다.
2. 루트 노드에 존재하는 값은 모든 노드의 값보다 가장 작기 때문에
   루트 노드의 값이 K값을 넘을 때까지 섞는 과정을 반복한다.
3. 섞는 과정을 반복한 후 더 이상 섞을 소스가 없거나, 1개 존재하지만 K를 넘기지 못하는 경우 -1,
4. 그 외의 경우 섞은 횟수를 반환한다.
